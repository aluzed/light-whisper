Built for simplicity

Global Hotkey

Alt+Space works from any application. No need to switch windows or open a separate tool.

100% Offline

All processing happens locally on your machine. Your voice data never leaves your computer.

Multi-Language

French, English, and 25+ languages with Parakeet. Automatic language detection included.

Auto-Paste

Transcribed text is automatically pasted into the active application via clipboard simulation.

Minimal UI

A tiny frameless overlay appears during recording. Settings are accessible from the system tray.

Lightweight

Built with Tauri and Rust. Small binary, low memory footprint, fast startup.

Two powerful engines

Choose the model that fits your needs — from 40 MB ultra-fast to high-accuracy multi-language.
